Svītrkodos glabātās informācijas nolasīšana vai kvadrātkodu izveidošana adrešu, kontaktu vai teksta kopīgošanai starp ierīcēm.

Atbalsta šādus svītrkodu veidus:

* UPC-A un UPC-E
* EAN-8 un EAN-13
* Code 39; 93; 128
* ITF
* Codabar
* RSS-14 (visi varianti)
* Kvadrātkods
* Data Matrix
* Aztec ('beta' laidiens)
* PDF 417 ('alpha' laidiens)
